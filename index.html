Question 2

Write a PHP program that collects input form the user through a textbox and checks whether the given number is an Armstrong number or not.
• If the number is indeed an Armstrong number, display all numbers in range form 1 to that specified number.(Note:0 is not an Armstrong number).
• Perform text validation, including checks for positive integer and general text input.

a2/armstrong.php
<!DOCTYPE html>
<html>

<head>
    <title>Armstrong Number Checker</title>
</head>

<body>

    <h2>Armstrong Number Checker</h2>

    <form method="post">

        Enter a number:
        <input type="number" name="number" required>

        <input type="submit" value="Check">

    </form>

    <?php

    function isArmstrong($num)
    {
        $original = $num;
        $sum = 0;
        $digits = strlen($num);

        while ($num > 0)
        {
            $digit = $num % 10;
            $sum += $digit ** $digits;
            $num = (int)($num / 10);
        }

        return $original == $sum;
    }

    if ($_SERVER["REQUEST_METHOD"] == "POST")
    {
        $input = $_POST["number"];

        if ($input < 0)
        {
            echo "<p>Please enter a positive number.</p>";
        }
        elseif ($input == 0)
        {
            echo "<p>0 is NOT an Armstrong Number.</p>";
        }
        else
        {
            if (isArmstrong($input))
            {
                echo "<p><b>$input is an Armstrong Number.</b></p>";

                echo "<p>Armstrong numbers from 1 to $input:</p>";

                for ($i = 1; $i <= $input; $i++)
                {
                    if (isArmstrong($i))
                    {
                        echo $i . " ";
                    }
                }
            }
            else
            {
                echo "<p><b>$input is NOT an Armstrong Number.</b></p>";
            }
        }
    }

    ?>

</body>

</html>

Question 4

Write a simple PHP program with a user interface for a mathematical calculator using HTML forms.
Perform division by zero operation with appropriate display of messages.
Perform text validation, special character validation for textboxes with appropriate display of messages.

a4/cal.php
<!DOCTYPE html>
<html>
<body>

<h2>PHP Calculator</h2>

<form method="post">

    <input type="text" name="a" placeholder="First Number" required>

    <select name="op">
        <option value="+">+</option>
        <option value="-">-</option>
        <option value="*">*</option>
        <option value="/">/</option>
    </select>

    <input type="text" name="b" placeholder="Second Number" required>

    <input type="submit" value="Calculate">

</form>

<?php

if($_SERVER["REQUEST_METHOD"]=="POST")
{
    $a = $_POST['a'];
    $b = $_POST['b'];
    $op = $_POST['op'];

    // Validate numbers
    if(!is_numeric($a) || !is_numeric($b))
    {
        echo "Enter valid numbers";
    }

    // Division by zero
    elseif($op=="/" && $b==0)
    {
        echo "Division by zero not allowed";
    }

    else
    {
        if($op=="+")
            $result = $a + $b;

        elseif($op=="-")
            $result = $a - $b;

        elseif($op=="*")
            $result = $a * $b;

        else
            $result = $a / $b;

        echo "<h3>Result = $result</h3>";
    }
}

?>

</body>
</html>

    <title>String Manipulation</title>
</head>

<body>

    <h2>String Manipulation</h2>

    <form method="post">

        Enter a string:

        <input type="text" name="input_string" required>

        <br><br>

        <input type="submit" name="length" value="Get Length">

        <input type="submit" name="reverse" value="Reverse">

        <input type="submit" name="uppercase" value="Uppercase">

        <input type="submit" name="lowercase" value="Lowercase">

        <input type="submit" name="replace" value="Replace">

        <input type="submit" name="palindrome" value="Check Palindrome">

        <input type="submit" name="shuffle" value="Shuffle">

        <input type="submit" name="count" value="Word Count">

    </form>

    if ($_SERVER["REQUEST_METHOD"] == "POST")
    {
        $str = $_POST["input_string"];

        if (isset($_POST["length"]))
        {
            echo "Length: " . strlen($str);
        }

        elseif (isset($_POST["reverse"]))
        {
            echo "Reverse: " . strrev($str);
        }

        elseif (isset($_POST["uppercase"]))
        {
            echo "Uppercase: " . strtoupper($str);
        }

        elseif (isset($_POST["lowercase"]))
        {
            echo "Lowercase: " . strtolower($str);
        }

        elseif (isset($_POST["replace"]))
        {
            echo "Replace: " . str_replace("a", "x", $str);
        }

        elseif (isset($_POST["palindrome"]))
        {

            {
                echo "The string is a palindrome.";
            }
            else
            {
                echo "The string is not a palindrome.";
            }
        }

        elseif (isset($_POST["shuffle"]))
        {
            echo "Shuffle: " . str_shuffle($str);
        }

        elseif (isset($_POST["count"]))
        {
            echo "Word Count: " . str_word_count($str);
        }
    }

Question 8

Write a PHP user interface program with an HTML form to input a string. Upon submission, it will display the number of times each word occurs, ignoring the distinction between capital and lowercase letters. It should also print the most and least used words. Additionally, include buttons for sorting data in ascending order and descending order.

a8/word.php
<!DOCTYPE html>
<html>

<head>
    <title>Word Frequency Analyzer</title>
</head>

<body>

    <h2>Word Frequency Analyzer</h2>

    <form method="post">

        Enter a string:

        <input
            type="text"
            name="text"
            required
            value="<?php echo $_POST['text'] ?? ''; ?>">

        <button type="submit">
            Analyze
        </button>

        <button type="submit" name="asc">
            Sort Ascending
        </button>

        <button type="submit" name="desc">
            Sort Descending
        </button>

    </form>

    <?php

    if ($_SERVER["REQUEST_METHOD"] == "POST")
    {
        $text = strtolower($_POST["text"]);

        $words = explode(" ", $text);

        $count = array_count_values($words);

        if (isset($_POST["asc"]))
        {
            asort($count);
        }

        elseif (isset($_POST["desc"]))
        {
            arsort($count);
        }

        echo "<h3>Word Frequencies:</h3>";

        foreach ($count as $word => $value)
        {
            echo "$word : $value times<br>";
        }

        $max = max($count);

        $min = min($count);

        $most = implode(", ", array_keys($count, $max));

        $least = implode(", ", array_keys($count, $min));

        echo "<br><b>Most used word:</b> $most ($max times)<br>";

        echo "<b>Least used word:</b> $least ($min times)";
    }

    ?>

</body>
</html>
